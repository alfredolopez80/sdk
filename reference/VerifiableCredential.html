(async) verify(resolveropt, compactProofopt, forceRevocationCheckopt, revocationAPIopt) → {Promise.<VerifiableCredentialVerificationResult>}

Verify a Verifiable Credential
Parameters:
Name Type Attributes Default Description
resolver object <optional>
 null Resolver for DIDs.
compactProof Boolean <optional>
 true Whether to compact the JSON-LD or not.
forceRevocationCheck Boolean <optional>
 true Whether to force revocation check or not. Warning, setting forceRevocationCheck to false can allow false positives when verifying revocable credentials.
revocationAPI object <optional>
 null An object representing a map. "revocation type -> revocation API". The API is used to check revocation status. For now, the object specifies the type as key and the value as the API, but the structure can change as we support more APIs there are more details associated with each API. Only Dock is supported as of now.
