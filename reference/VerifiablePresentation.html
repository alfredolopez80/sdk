Verify a Verifiable Presentation
Parameters:
Name Type Default Description
challenge string proof challenge Required.
domain string proof domain (optional)
resolver Resolver Resolver to resolve the issuer DID (optional)
compactProof Boolean true Whether to compact the JSON-LD or not.
forceRevocationCheck Boolean true Whether to force revocation check or not. Warning, setting forceRevocationCheck to false can allow false positives when verifying revocable credentials.
revocationAPI object An object representing a map. "revocation type -> revocation API". The API is used to check revocation status. For now, the object specifies the type as key and the value as the API, but the structure can change as we support more APIs there are more details associated with each API. Only Dock is supported as of now.
